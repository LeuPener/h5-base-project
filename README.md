# h5-base-project
 📱 Mobile terminal base project
 🍭 Support Vue-v2.7.0
 🎉 Support Less
 🎨 theme with vant
 🌍 Built-in i18n solution

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm build
```

### Lints and fixes files
